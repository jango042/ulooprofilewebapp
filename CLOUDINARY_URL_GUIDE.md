# Cloudinary URL Format Guide

## Which URL Format Should You Use?

Based on your Cloudinary URL structure, here's what to use:

### ✅ **Recommended: Use Public IDs Only**

**Don't use:** Full URLs with version numbers
```
❌ https://res.cloudinary.com/jango042/image/upload/v1768998239/excel_camp_2025/DSC_1441_azvyn5.jpg
```

**Use instead:** Just the public ID
```
✅ excel_camp_2025/DSC_1441_azvyn5
```

The code will automatically:
- Generate **thumbnails** for the gallery grid (400x400px, optimized)
- Use **full-size optimized** images for the lightbox view
- Apply automatic quality and format optimization

### URL Formats Explained

#### 1. **Thumbnail** (For Gallery Grid)
- **Use:** Small, optimized images
- **Format:** `w_400,h_400,c_fill,q_auto`
- **Purpose:** Fast loading in grid view
- **Auto-generated by code**

#### 2. **Original/Full Size** (For Lightbox)
- **Use:** Full resolution, optimized
- **Format:** `q_auto,f_auto` (quality and format auto)
- **Purpose:** High quality when clicked
- **Auto-generated by code**

#### 3. **Watermarked Versions**
- **Don't use:** Unless you specifically want watermarks
- **Skip these:** They're usually for protection/preview

#### 4. **Circle/Custom Shapes**
- **Don't use:** Unless you want circular images
- **Use standard:** Rectangular images work best for galleries

## Quick Setup Steps

### Option 1: Use Public IDs (Recommended)

In `src/data/campImages.js`:
```javascript
export const campImages = [
  { publicId: 'excel_camp_2025/DSC_1441_azvyn5', tags: ['day1'] },
  { publicId: 'excel_camp_2025/DSC_1442_xyz123', tags: ['day1', 'activities'] },
  // ... add all your images
]
```

### Option 2: Use Full URLs (Auto-extracted)

You can also paste full URLs - the code will extract the public ID automatically:
```javascript
export const campImages = [
  { publicId: 'https://res.cloudinary.com/jango042/image/upload/v1768998239/excel_camp_2025/DSC_1441_azvyn5.jpg', tags: ['day1'] },
  // ... the code will extract: excel_camp_2025/DSC_1441_azvyn5
]
```

## Getting All Your Public IDs

### Method 1: From Cloudinary Dashboard
1. Go to Media Library
2. Click on each image
3. Copy the "Public ID" field
4. Add to `campImages.js`

### Method 2: Bulk Export (If Available)
1. In Media Library, select all images
2. Use Cloudinary's export feature
